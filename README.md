# CodeVault

A MERN-based platform for tracking DSA problems, organizing coding notes, and monitoring learning progress.

## Planned Features

- User Authentication
- DSA Problem Tracking
- Topic-wise Organization
- Personal Notes Management
- Bookmark Important Problems
- Progress Dashboard

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB

## Project Status

🚧 Currently under development

## Folder Structure

CodeVault/
├── client/
├── server/
└── docs/

## Goal

To help students organize their coding journey, maintain notes, and track DSA progress efficiently.
